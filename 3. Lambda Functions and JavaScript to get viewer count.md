## Building a Viewer Counter with AWS Lambda and JavaScript

*This document summarizes key concepts and steps involved in creating a dynamic viewer counter for a website using AWS Lambda and JavaScript, based on "AWS Cloud Resume Challenge - Lambda Functions and JavaScript to get viewer count"

**Core Components:**

- **DynamoDB Table:** Stores the view count. A table named "resume-challenge" is used with an item having ID "0" and a "views" attribute holding the count.
- **Lambda Function (Python):** Acts as the API endpoint.
	- Uses boto3 library to interact with DynamoDB.
	- Fetches the current view count from the table.
	- Increments the count by 1.
	- Updates the DynamoDB table with the new count.
	- Configured with a function URL and CORS enabled for website access.

*We'll be using python as our programming language for the API and we'll be using JavaScript in our front-end website to display the viewer counter which we'll be getting from the dynamodb table"

- **Website (HTML and JavaScript):**HTML includes a designated element (e.g., <div class="counter-number">) to display the view count.
- JavaScript fetches the count from the Lambda function URL using fetch.
- Updates the designated HTML element with the retrieved count.

**Workflow:**

1. Website loads, triggering the JavaScript code.
2. JavaScript sends a request to the Lambda function URL.
3. Lambda function retrieves the current count from DynamoDB, increments it, and updates the table.
4. Lambda function returns the updated count to the website.
5. JavaScript updates the designated HTML element with the received count.

Key Considerations: 

- Function URL: Enables direct access to the Lambda function from the website.
- CORS Configuration: Allows cross-origin requests from the website domain to the Lambda function.
- Security: Restricting access to the Lambda function URL to authorized origins is crucial.

"I recommend to specify the origin to be your url of the website and don't allow anyone to just curl your Lambda function"

**Implementation Highlights:**

- Concise Python code leveraging boto3 for efficient DynamoDB interaction.
- Simple JavaScript implementation using fetch and DOM manipulation for dynamic updates.
- Practical demonstration of integrating serverless functions with front-end development.
